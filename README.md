# Admin Dashboard - HTML & CSS Practice

This is a simple Admin Dashboard project I built to practice my **HTML** and **CSS** skills. I am currently learning web development at **SMIT (Saylani Mass IT Training)**.

---

## About the Project

I created this project to practice how to build a website with multiple pages. It has a login page and a dashboard where you can see customers, orders, and products.

**Note:** This is a basic project for practice. It only uses HTML and CSS to show how a dashboard looks. It doesn't have a real backend or security yet.

---

## CSS Features I Used

In this project, I practiced many important CSS basics:

- **Fixed Layout:** I made the top header and side menu stay in place while you scroll.
- **Sidebar Menu:** I built a side menu that stays on the left side of the screen.
- **Cards:** I created white boxes (cards) with shadows to show important information.
- **Hover Effects:** I added smooth effects so that buttons and links change color or grow slightly when you put your mouse over them.
- **Tables:** I styled tables to make them look clean and easy to read.
- **Colors:** I used different colors to show if an order is "Pending," "Complete," or "Cancelled."
- **Round Images:** I learned how to make profile pictures round using borders.

---

## 📂 Files in this Project

- `index.html` & `login.css`: The first page you see (Login).
- `admin-dashboard.html` & `admin-style.css`: The main dashboard page.
- `customers.html`, `orders.html`, `products.html`: Pages to show different data.
- `settings.html` & `settings.css`: A page to change settings.
